# Scatter GDM greeter overrides

What this ships:

- `90-scatter` → `/etc/dconf/db/gdm.d/90-scatter` — dconf keyfile that
  forces dark color scheme, yellow accent (closest named to amber),
  black background, and swaps the distributor logo to the Scatter mark.
- `greeter-logo.png` → `/usr/share/scatter/greeter-logo.png` — Scatter
  distributor mark, generated by `../generate_assets.py`.
- `avatar.png` → `/var/lib/AccountsService/icons/<user>` — user avatar,
  generated by `../generate_assets.py`.

Known gaps (tier-2 work):

- The "Ubuntu" wordmark that appears at the greeter bottom is drawn by
  gnome-shell's loginDialog, compiled into `libgnome-shell.so`. Killing
  it requires rebuilding the shell gresource, deferred.
- Yaru-yellow accent ≠ `#ffb800` exactly. Fully matching amber requires
  shipping a custom shell theme gresource.
